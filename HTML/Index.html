Ciber Seguriad

La ciberseguridad es el conjunto de tareas llevadas a cabo para proteger archivos digitales, así como para acabar con cualquier amenaza que pueda interferir en el acceso no deseado a información confidencial.

Cuando se habla de ciber seguridad se dice que se suele emplear para proteger datos, pero la ciberseguridad solo es el conocimiento, se puede aplicar para proteger, o para atacar.

A los atacantes se les suele llamar hackers de sombrero negro, y a los defensores de hackers de sombero blanco

También hay que ser onscientes de que un hacker puede desde quitarte una cuenta de instagram, hasta meterse en la red del gobierno y sacar información confidencial
